# Getting started:

<a name="desc"></a>
## Installation:

Opencv:
   
       pip install regex
       
Docx:
 
        pip install python-docx
        
Os:
        
        pip install os-sys

## Usage:
```
1/ Add files that you want to convert into folder "source"

2/ Run "convert.py"

3/ The files are now converted and located in the "output" folder.
```
